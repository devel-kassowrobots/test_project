◆ getKRService()

Object com.kassowrobots.api.app.KRContext.getKRService ( String  aName)

Returns the handle to a KR-level service by name. The class of the returned object varies by the requested name. Currently available names are:

TEACHING_MANAGER_SERVICE ("teaching_manager")
A TeachingManager for subscribing teaching events which let you to override behaviour of default teaching handler.
SELECTION_MANAGER_SERVICE ("selection_manager")
A SelectionManager for controlling and monitoring selection of variables.
DRAG_MANAGER_SERVICE ("drag_manager")
A DragManager for subscribing drag events by views.

Note: KR services obtained via this API are available (non null) only if the CBun app is launched in the host (Boson) application.

Parameters
aNameThe name of the desired service.
Returns
The service or null if the service is not available
