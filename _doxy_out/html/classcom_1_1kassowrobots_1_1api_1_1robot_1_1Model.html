Frame com.kassowrobots.api.robot.Model.getFrame ( FrameID  aFrameID,
FrameID  aRefFrameID 
)

Retrieves system frame for given system ID and reference system ID. Currently available frames are:

FrameID.WORLD ("/sys/model/robot/frames/FRMWORLD")
Identifies the world frame.
FrameID.BASE ("/sys/model/robot/frames/FRMRBASE")
Identifies the robot base frame.
FrameID.LINK1 ("/sys/model/robot/frames/FRMRLINK1")
Identifies the robot link 1 frame.
FrameID.LINK2 ("/sys/model/robot/frames/FRMRLINK2")
Identifies the robot link 2 frame.
FrameID.LINK3 ("/sys/model/robot/frames/FRMRLINK3")
Identifies the robot link 3 frame.
FrameID.LINK4 ("/sys/model/robot/frames/FRMRLINK4")
Identifies the robot link 4 frame.
FrameID.LINK5 ("/sys/model/robot/frames/FRMRLINK5")
Identifies the robot link 5 frame.
FrameID.LINK6 ("/sys/model/robot/frames/FRMRLINK6")
Identifies the robot link 6 frame.
FrameID.LINK7 ("/sys/model/robot/frames/FRMRLINK7")
Identifies the robot link 7 frame.
FrameID.FLANGE ("/sys/model/robot/frames/FRMRFLNG")
Identifies the TFC (tool flange center) frame.
FrameID.TCP ("/sys/model/robot/frames/FRMTCP")
Identifies the TCP (tool center point) frame.
FrameID.TOOL ("/sys/model/robot/frames/FRMTOOL")
Identifies the tool frame (for tool load configuration).
FrameID.PAYLOAD ("/sys/model/robot/frames/FRMPAYLD")
Identifies the payload frame (for payload configuration).
FrameID.TARGET ("/sys/model/robot/frames/FRMTRGT")
Identifies the last target frame, ie. the target coordinates of the last linear MOVE command.
FrameID.XRF ("/sys/model/robot/frames/FRMXRF")
Identifies the XRF frame.
FrameID.ARTO ("/sys/model/robot/frames/FRMARTO")
Identifies the ARTO frame.
Parameters
aFrameIDSystem ID of requested frame.
aRefFrameIDSystem ID of reference frame, ie. another system frame to be used as a reference frame.
Returns
Frame for given system ID and reference ID or null, if such system value does not exist.

JSVector com.kassowrobots.api.robot.Model.getJoints ( String  aSysId)

Retrieves system joints vector for given system ID. Currently available system joints vectors are:

SystemID.JOINTS_ANGLE ("/sys/model/robot/joints/JNTSANG")
Identifies the joints angle (position) vector.
SystemID.JOINTS_VELOCITY ("/sys/model/robot/joints/JNTSVEL")
Identifies the joints velocity vector.
SystemID.JOINTS_ACCELERATION ("/sys/model/robot/joints/JNTSACC")
Identifies the joints acceleration vector.
SystemID.JOINTS_TORQUE ("/sys/model/robot/joints/JNTSTRQ")
Identifies the joints torque vector.
Parameters
aSysIdSystem ID of requested joints vector.
Returns
Joints vector for given system ID or null, if such system joints vector does not exist.
