Detailed Description

Allows to construct anonymous variable. Such variable is not registered in the var register, but it exists only locally either withing the UI (for example in the VarDropBox) or in another variable (for example as a reference of some pose).

Parameters
<T>Data type of the variable (such as Number, RobotPose or Load).
