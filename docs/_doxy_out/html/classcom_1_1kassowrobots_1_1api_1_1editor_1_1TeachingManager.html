Detailed Description

Allows subscription of teaching events to override behaviour of default teaching intent handler. This class cannot be instantiated, instead its instance may be returned by KRContext.getKRService(String) method.

void com.kassowrobots.api.editor.TeachingManager.attach ( CustomTeachingHandler  aHandler)

Attaches new custom teaching handler to this teaching manager. This teaching handler will be invoked on every teaching intent unless the handler is detached. If any custom teaching handler is already attached, this request is ignored.

Parameters
aHandlerCustom teaching handler to be attached.
