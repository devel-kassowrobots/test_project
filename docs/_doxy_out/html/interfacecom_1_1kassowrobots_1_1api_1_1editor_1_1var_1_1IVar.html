◆ PROPERTY_SCOPE

String com.kassowrobots.api.editor.var.IVar< T >.PROPERTY_SCOPE = "scope"

Property name constant to be used as an argument of getProperty(String): Provides access to the variable scope:

SCOPE_ANONYMOUS ("anonymous")
Signals a variable that is not registered in any var register (ie. it exists only locally in the UI or in some other data type).
SCOPE_LOCAL ("local")
Signals a variable that is registered in program var register and its value is not shared among the sequences.
SCOPE_GLOBAL ("global")
Signals a variable that is registered in program var register and its value is shared among the sequences.
SCOPE_PERSISTENT ("persistent")
Signals a variable that is registered in workcell var register and its value is persistent across various program executions.

Note: The scope is returned only for "custom" variables.
